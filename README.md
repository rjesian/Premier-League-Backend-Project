Premier Zone – Premier League Players & Teams REST API

Premier Zone is a backend project built with Java and Spring Boot that models the Premier League’s players and teams. It demonstrates clean backend architecture and the ability to manage real-world data with dynamic queries, filtering, and database integration.

Project Overview

This project focuses on creating a robust, structured backend for managing Premier League players. It includes:

Full CRUD operations for players

Dynamic filtering by team, position, name, and nation

A clear Controller → Service → Repository design pattern

Integration with PostgreSQL for persistent storage

It’s designed to highlight backend engineering skills, such as data handling, API design, and logical structuring, rather than frontend features.

Key Features

Player Management: Add, update, delete, and retrieve player information.

Dynamic Filtering: Search players by multiple parameters (team, position, nation).

Clean Architecture: Separation of concerns ensures maintainable and scalable code.

Database Integration: Uses PostgreSQL to persist player data.

RESTful API: Built to handle real-world use cases for managing sports data.

Tech Stack

Language: Java 21

Framework: Spring Boot

Database: PostgreSQL

Build Tools: Maven
